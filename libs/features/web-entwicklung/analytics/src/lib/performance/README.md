# Performance (PageSpeed Insights) Slice

In diesem Verzeichnis liegt die gekapselte Logik zur Geschwindigkeitsmessung.

## Struktur
- `ui/`: Gauge-Charts (z.B. 95/100 Score) und Ampel-Systeme für Core Web Vitals.
- `data-access/`: Asynchrone API-Anfragen an die Google PageSpeed Schnittstelle zur Erfassung echter CrUX-Felddaten und simulierter Lighthouse-Werte.
